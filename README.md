# Bienvenido a mi repositorio de guías de estudio

Este repositorio es mi espacio personal para guardar y organizar los fundamentos de diversos lenguajes y tecnologías informáticas. Aquí puedo encontrar rápidamente conceptos o teorías relevantes cuando los necesite.

## Estructura del repositorio

He decidido mantener una estructura sencilla:
- Cada tecnología o lenguaje tiene su propia carpeta.
- Dentro de cada carpeta, guardo archivos README llamados `module-1.md`, `module-2.md`, y así sucesivamente.

## Recursos recomendados

Para complementar estas guías, aquí incluyo algunos recursos oficiales y altamente recomendados:

### Libros
- "Clean Code" de Robert C. Martin: Una guía esencial para escribir código limpio y mantenible.
- "The Pragmatic Programmer" de Andrew Hunt y David Thomas: Consejos prácticos para mejorar como desarrollador.
- "Introduction to the Theory of Computation" de Michael Sipser: Ideal para profundizar en fundamentos teóricos.

### Páginas web
- [MDN Web Docs](https://developer.mozilla.org): Documentación oficial para tecnologías web.
- [W3Schools](https://www.w3schools.com): Tutoriales y referencias para aprender tecnologías web.
- [GeeksforGeeks](https://www.geeksforgeeks.org): Recursos y explicaciones sobre algoritmos y estructuras de datos.
- [Stack Overflow](https://stackoverflow.com): Comunidad para resolver dudas y aprender de otros desarrolladores.
