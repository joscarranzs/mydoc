# Introducción a la programación

## ¿Qué es la programación?

La programación es el proceso mediante el cual se diseña, escribe y verifica un conjunto de instrucciones que una computadora debe ejecutar para realizar una tarea específica. Estas instrucciones se escriben en un **lenguaje de programación**, un lenguaje formal diseñado para comunicar órdenes a una máquina.

Un programa informático no es más que una secuencia de pasos lógicos que transforman una entrada en una salida. La computadora ejecuta estos pasos de manera literal, rápida y sin desviaciones, por lo que las instrucciones deben ser precisas, inequívocas y estar correctamente ordenadas.

La programación permite automatizar procesos, crear aplicaciones, analizar datos, controlar hardware y resolver problemas complejos mediante la descomposición de estos en pasos elementales.

## Ejemplo

A continuación se presenta un programa sencillo escrito en pseudocódigo. Su función es sumar dos números y mostrar el resultado:

```
INICIO
    LEER numero1
    LEER numero2
    suma = numero1 + numero2
    ESCRIBIR "La suma es: " + suma
FIN
```

**Explicación del ejemplo:**

| Línea | Acción |
|---|---|
| `INICIO` | Marca el punto de entrada del programa |
| `LEER numero1` | Obtiene el primer valor desde el teclado |
| `LEER numero2` | Obtiene el segundo valor desde el teclado |
| `suma = numero1 + numero2` | Calcula la suma y la almacena en una variable |
| `MOSTRAR ...` | Imprime el resultado en la pantalla |
| `FIN` | Marca el punto de finalización del programa |

Aunque el pseudocódigo no es ejecutable, ilustra la estructura fundamental de cualquier programa: entrada de datos, procesamiento y salida de resultados.

## Ejercicio

Escriba en pseudocódigo un programa que lea dos números, multiplique el primero por el segundo y muestre el resultado en pantalla.

Utilice la misma estructura del ejemplo anterior: `INICIO`, `LEER`, operación aritmética, `MOSTRAR` y `FIN`.

> *Nota: no es necesario utilizar ningún lenguaje de programación real. El propósito es familiarizarse con la secuencia lógica de instrucciones.*
